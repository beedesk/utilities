## Utilities JS
=========

### A Humble Collection of JavaScript Utilities That You Don't Want to Write Again

#### `Dates`
* `toISOString`
* `fromISOString`

#### `Matches`
* `match`
* `find`
* `findOnce`

#### `Arrays`
* `intersect`
* `extract`
* `contains`

#### `URLs`, `HashSearch`
* parse URLs
* parse Search parts (?key=value&)

#### `Strings`
* quotes
* `trim`
* `fill`
* etc.

#### `Hashs` (aka: associates map, or simply objects)
* `isEquals` -- deep compare
* `simplify`
* `has`
* `apply`
* etc.

#### `Binder`
* `bind`
* `unbind`
* `trigger`

#### `Base64`
* `encode`
* `decode`
